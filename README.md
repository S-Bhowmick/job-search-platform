# Job Search Platform

## Description
This is a job search platform where users can search for job listings, post jobs, and log in to manage their profiles. The platform uses Node.js for the backend and MongoDB for storing job listings and user data.

## Features
- **Search Jobs**: Search job listings by keywords and location.
- **Post Jobs**: Employers can post job listings with specific job details.
- **Authentication**: Users can register, login, and manage their profiles.
- **Job Listings**: Display dynamic job listings fetched from MongoDB.

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript (Vanilla)
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: JSON Web Token (JWT)
